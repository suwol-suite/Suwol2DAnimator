# Animation Timelines v8 Sample

This sample focuses on the v8 animation timeline additions:

- attachment timeline on `weapon_slot`: `sword` to `axe` to hidden
- draw order timeline swapping `weapon_slot` and `arm_slot`
- slot color/alpha timeline on `body_slot` and `weapon_slot`
- event timeline with `footstep` and `attack`

Import this sample through Package Manager, then either:

- select `sample_animation_timelines.suwol2d` and inspect the generated prefab/report, or
- run `Tools/Suwol2D/Create Animation Timelines v8 Demo`.

The `.suwol2d` and `.suwol2d.json` files intentionally contain the same JSON data.
