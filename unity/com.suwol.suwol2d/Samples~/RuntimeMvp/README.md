# Runtime MVP v0 Sample

Import this sample from Unity Package Manager, then run:

```text
Tools/Suwol2D/Create Runtime MVP Demo
```

The menu creates a `Suwol2DCharacter` GameObject and tries to assign:

- `sample_character.suwol2d.json`
- `Textures/body.png`
- `Textures/arm.png`
- `Assets/Suwol2D Runtime MVP/Suwol2DDefault.mat`

Enter Play Mode to see the `idle` animation. Use the `Suwol2DCharacter` inspector buttons in Play Mode to switch between `idle` and `walk`.

If automatic assignment fails, select the demo GameObject and assign the same assets manually in the `Suwol2DCharacter` inspector.
