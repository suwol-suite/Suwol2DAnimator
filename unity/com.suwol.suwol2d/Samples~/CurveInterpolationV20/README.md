# Curve Interpolation v20 Sample

This sample demonstrates keyframe interpolation presets supported by Suwol 2D Animator v20.

- `idle` keeps a simple looping pose with slot color easing.
- `walk` preserves the existing timeline editing sample behavior.
- `curve_test` includes `stepped`, `linear`, `easeIn`, `easeOut`, and `easeInOut` keys across bone, slot color, and deform timelines.

Discrete attachment, draw order, and event timelines do not use interpolation.
